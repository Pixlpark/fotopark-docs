# 7. Печать альбомов
## Автоматическая печать
* После проверки всех альбомов можно перейти к созданию заказа в партнерской типографии, чъя продукция была выбрана для проекта. Для этого надо открыть во вкладку "__Настройки__" и выполнить следующие действия:
    + В блоке "__Заказы__" Нажать кнопку "__Создать__".
    + В открывшемся модальном окне выбрать участников, изменить количество альбомов через кнопку "шестиренка", выбрать доставку и нажать кнопку "__Создать__". Обращаем внимание, что выбрать можно только тех учеников, чьи альбомы были утверждены.
    + Если возникает потребность в допечатке тиража, то можно содать абсолютно новый заказ, либо добавить позицией к существующему, если он еще не ушел в печать.
    + Для поступаления заказа на производство необходимо нажать кнопку "__Оплатить__" - сервис переведет на странице оплаты сайта типографии. Сам проект при этом получит статус "__В печати__".
    + После выполнения заказа он будет передан в транспортную компанию, а вам на эл. почту придет сообщение для отслеживания отправления.
    + После выдачи заказа проект получит финальный статус - "__Выдан__".

## Самостоятельная печать
* Помимо работы с партнерской типографией сервис допускает печать заказа в любой другой, скачав подготовленные макеты альбомов или фотографий.
* Для этого необходимо сперва создать обычный заказ, а затем, не оплачивая, его скачать файлы для печати. Такой подход может быть удобен, если вы планируете альбомы напечатать в партнерской типографии, а фотографии - в локальном фотоцентре.

## Бэкпринт для фотографий
* __Бэкпринт__ - это короткая техническая информация, которая наносится на обратную сторону фотографии. Бэкпринт может содержать данные о номере заказа, имени файла, дате печати, настройках цветокоррекции, плотности, а также название производителя фотобумаги. Это позволяет идентифицировать отпечатки и отслеживать параметры печати. 
* При заказе фотопечати в типографии "__Фабрика Фотокниги__" в качестве бэкпринта мы размещаем информацию, которая в последствии поможет рассортировать снимки по ученикам. 
* В общем случае формат бэкпринта из 12 символов такой: ```название_группы:номер_ученика:номер_фото```, где:
    + __название_группы__ - первые четыре символа преоразованного в латиницу названия группы (или класса), в которой состоит ученик (например: 1A, 10D).
    + __номер_ученика__ - порядковый номер ученика в заказе, состоящий из четырех символов (например: 0001, 0012, 0123, 1234). Порядковый номер каждого ученика можно посмотреть в детализации заказа, нажав на его номер.
    + __номер_фото__ - порядковый номер фото у ученика, состоящий из двух символов (например: 01, 12).

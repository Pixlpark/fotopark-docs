# 2. Отбор фотографий
* Следующим этапом после настройки проекта является отбор фотографий для формирования дизайнов. Это может сделать фотограф/координатор или сам ученик. Набор необходимых фотографий определяется используемым дизайном. В панели управления работа по выбору фотографий ведется во вкладке “__Отбор фото__”.
* ![](../_media/projects/selection-photos.png)
* В общем случае интерфейс раздела содержит несколько вкладок:
    + __Фото учеников/педагогов__ - это портретные, репортажные и групповые снимки на персональных разворотах альбомов участников.
    + __Фото на общих разворотах__ - это репортажные и групповые снимки на общих разворотах каждого дизайна. Вкладка будет отсутствовать, если в дизайне все развороты персональные.
    + __Фото по умолчанию__ - это репортажные и групповые снимки на персональных разворотах каждого дизайна, которые будут использоваться в двух случаях:
        + Согласно настройкам доступа в ЛК у ученика запрещен выбор персонлаьных фото.
        + Согласно разделу "Настройка" включена функция автодобора персонлаьных фотографий.
* Для отбора фотографий необходимо:
    1. Нажать кнопку "__Выбрать фото__" у интересующего блока, откроется модальное окно.
    2. Загрузить собственные снимки, если это разрешено согласно разделу "Настройка".
    2. Выставить необходимый фильтр слева и, если требуется, сгруппировать снимки.
    3. Выделить понравишиемся фотографии и нажать кнопку "__Сохранить__".
